# Instad Venue — Venue Owner Management App

The venue owner app in the Instad ecosystem. Sports venue owners create and manage listings, set availability, handle bookings, and communicate with customers.

![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?logo=dart&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?logo=firebase&logoColor=black)

## Features

- Venue listing creation and management (photos, descriptions, amenities)
- Availability calendar with time slot configuration
- Booking management (view, confirm, decline)
- Customer communication tools
- Revenue and booking analytics

## Tech Stack

Flutter, Dart, Firebase (Auth, Firestore, Storage, Functions)

## Running

```bash
git clone https://github.com/thisisyoussef/instad_venue.git
cd instad_venue && flutter pub get && flutter run
```

Requires Firebase config files.
